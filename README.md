# ZephiPay P2P Demo

Phase 1 prototype for ZephiPay's peer-to-peer payment flow.

## Current features
- User balance simulation
- Transfers by user name
- Failed transfer handling
- Transaction history
- Transaction IDs
- Timestamps
- Failure reasons

## Tech stack
- Python for behavior modeling
- TypeScript for product-layer simulation

## Goal
Design a payment experience that feels instant, obvious, and effortless before connecting to blockchain infrastructure.
